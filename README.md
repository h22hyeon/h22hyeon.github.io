# Personal Academic Website

This repository is a GitHub Pages website built with Jekyll (GitHub Pages compatible).
It uses the Minimal Mistakes theme as a remote theme and applies a “McKinsey-like” minimal typographic layer.

## 1) Quick Start (GitHub Pages)
1. Create a repository named: `YOUR_GITHUB_USERNAME.github.io`
2. Push these files to `main`
3. In GitHub repo: Settings → Pages → Deploy from branch → Branch `main` / root
4. Visit: https://YOUR_GITHUB_USERNAME.github.io

## 2) Local Preview
Requirements: Ruby + Bundler

```bash
bundle install
bundle exec jekyll serve
